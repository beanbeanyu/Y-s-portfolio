p5.js Assignments

  1. Project 1: Lost and Found

    Created a drawing using p5.js of my partner's lost wallet in NYC.

  2. Project 2: Face Generator

    By clicking the mouse or using the keyboard, the background and the protagonist in the work can change in sequence.

  3. Project 3: Experimental Clock

    This is a clock that shows how your mood changes throughout the day. The circle moving to the right represents the 24 hours of a day. The circles moving up and down represent different emotions. Blue for unhappy, green for general, red for happy mood.

  4. Project 4: Exquisite Corpse

    I coded character designs from three different friends of mine and put them together.

  5. Project 5: Optical Illusion

    By moving the mouse, the color and shape size can be controlled, thus forming a visual illusion effect.

  6. Project 6: Data Portrait

    A visualization of my use of different types of apps on my phone over a week.

  7. Project 7: Autobiographical Game

    This is a 2D fishing game. In the game, players need to use superb fishing skills to complete the set goals and avoid bombs.
